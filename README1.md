# Customer Shopping Behavior Analysis

## Project Overview
The objective of this project is to analyze customer shopping behavior and identify purchasing patterns, customer preferences, and revenue trends. The insights generated can help businesses make data-driven decisions related to marketing, customer retention, and sales optimization.

## Tools Used
- MS Excel
- Python
- PostgreSQL
- Power BI
- GitHub

## Features
- Data cleaning and preprocessing using Python.
- Exploratory Data Analysis (EDA) to identify customer purchasing patterns.
- Analysis of customer demographics and shopping behavior.
- Visualization of sales trends and product performance.
- Interactive Power BI dashboard with filters and slicers.

## Dataset
Customer Shopping Behavior dataset containing •	Customer ID 
•	Age 
•	Gender 
•	Item Purchased 
•	Category 
•	Purchase Amount 
•	Location 
•	Season 
•	Review Rating 
•	Subscription Status 
•	Shipping Type 
•	Discount Applied 
•	Promo Code Used 
•	Previous Purchases 
•	Payment Method 
•	Frequency of Purchases
